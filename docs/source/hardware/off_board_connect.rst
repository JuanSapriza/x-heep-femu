Off-board connectivity
======================

.. image:: ../images/pinout.svg
   :width: 600

Our FPGA platform offers off-board connectivity to allow the user to extend the design with his own sensors' boards. 

The available external interfaces are:

- ``SPI``
- ``I2C``
- ``GPIO[31:2]``
- ``reset``
